    Ինստալյացիա

    Այս էջը ցույց կտա Eiger-ի տեղադրման քայլերը յուրաքանչյուր օպերացիոն համակարգի համար

    Windows

    Գնացեք ռեպո-ի թողարկումների էջ և ներբեռնեք վերջին թողարկման eiger-win.zip ֆայլը:

    Եթե ​​դեռ լեզվի կայուն թողարկումներ չկան, խնդրում ենք դիմել «Կառուցում աղբյուրից».

    Միջավայրային փոփոխականի ավելացում (Windows)

    setx PATH "%PATH%;C:\ուղի\դեպի\eiger-win"

    macOS

    խնդրում ենք դիմել «Կառուցում աղբյուրից».

    Linux
