# __:fontawesome-solid-users: Նպաստել նախագծին__

Մենք փնտրում ենք ներդրողներ, որոնք կօգնեն մեզ զարգացնել լեզուն: Եթե ​​դուք հետաքրքրված եք, խնդրում ենք ներդրումներ կատարել `eigerproject/eigerlang`-ում:

## :fontawesome-solid-question: Ինչպես նպաստել նախագծին
1. Fork արեք repository-ն
    - Բացեք `eigerproject/eigerlang`
    - Սեղմեք `Fork` կոճակը էջի վերևի աջ կողմում
2. Կլոնավորեք repository-ն
    - Կլոնավորեք Ձեր fork արած repository-ն Ձեր համակարգչի վրա
    ```
    git clone https://github.com/YOUR-USERNAME/eigerlang.git
    ```
    - Բացեք կլոնավորած թղթապանակը
    ```
    cd eigerlang
    ```
3. Ստեղծեք նոր ճյուղ
    - Ստեղծեք նոր ճյուղ Ձեր փոփոխությունների համար
    ```
    git checkout -b my-feature-branch
    ```
4. Կատարեք Փոփոխություններ
    - Կատարեք Ձեր փոփոխությունները կամ լրացումները կոդի բազայում
    - Համոզվեք, որ Ձեր կոդը հետևում է նախագծի կոդավորման չափանիշներին:
5. Commit արեք փոփոխությունները
    - Ավելացրեք Ձեր փոխած ֆայլերը
    ```
    git add .
    ```
    - Commit արեք Ձեր փոփոխությունները բովանդակալից message-ով
    ```
    git commit -m "Նկարագրեք ձեր փոփոխություններն այստեղ"
    ```
6. Push արեք Փոփոխությունները ճյուղին
    ```
    git push origin my-feature-branch
    ```
7. Ստեղծեք PR (Pull Request)
    - Բացեք `eigerproject/eigerlang`
    - Սեղմեք "Compare & Pull Request"
    - Տրամադրեք հստակ վերնագիր ձեր Pull Request-ի համար
    - Նկարագրեք ձեր կատարած փոփոխությունները և ինչու են դրանք անհրաժեշտ
    - Submit արեք Pull Request-ը

## :fontawesome-solid-check: Ներդրումների ուղեցույցներ

- **Կոդի ոճ**՝ Համոզվեք, որ ձեր կոդը համապատասխանում է նախագծում օգտագործվող կոդավորման ստանդարտներին: Գործարկեք formatter-ը (`Ctrl+K Ctrl+D` Visual Studio-ի համար) ըստ պահանջի:
- **Թեստավորում**՝ Գրեք թեստեր ձեր փոփոխությունների համար և համոզվեք, որ բոլոր առկա թեստերն անցնեն: **Միավոր թեստերը շուտով կլինեն**
- **Դոկումենտացիա**՝ Թարմացրեք դոկումենտացիան, որպեսզի արտացոլեն ձեր փոփոխությունները, եթե պետք է: Համոզվեք, որ փոփոխությունները հստակ և հակիրճ են:
- **Commit հաղորդագրություններ**՝ Օգտագործեք բովանդակալից և նկարագրական հաղորդագրություններ:
