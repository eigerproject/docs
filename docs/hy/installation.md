# __:fontawesome-solid-circle-down: Ինստալյացիա__

Այս էջը ցույց կտա Eiger-ի տեղադրման քայլերը յուրաքանչյուր օպերացիոն համակարգի համար

---

## :fontawesome-brands-windows: Windows
Գնացեք ռեպո-ի [թողարկումների էջ](https://github.com/eigerproject/eigerlang/releases) և ներբեռնեք վերջին թողարկման `eiger-win.zip` ֆայլը:
> __Եթե ​​դեռ լեզվի կայուն թողարկումներ չկան, խնդրում ենք դիմել [«Կառուցում աղբյուրից»](#_2).__
### Միջավայրային փոփոխականի ավելացում (Windows)
```batch
setx PATH "%PATH%;C:\ուղի\դեպի\eiger-win"
```
